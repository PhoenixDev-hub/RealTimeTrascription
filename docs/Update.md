# Ideias para Evolução do DualLibras.AI

## Inteligência Artificial

- Utilizar o plano de aula como contexto para reduzir erros na transcrição.
- Permitir upload de PDF, DOCX, PPT e slides para enriquecer o contexto da IA.
- Criar um glossário personalizado para cada disciplina.
- Memória de contexto durante a aula para compreender referências anteriores.
- Correção automática de termos técnicos.
- Resumos automáticos ao final da aula.
- Extração automática de palavras-chave.
- Pesquisa inteligente nas transcrições.
- Identificação automática de quem está falando.
- IA especializada por área (Matemática, Biologia, Direito, Medicina, Programação...).

---

## Plataforma

- Landing Page apresentando o problema, solução, tecnologias e impacto social.
- Sistema de cadastro para professores, alunos e administradores.
- Dashboard para professores.
- Dashboard para alunos.
- Histórico de aulas.
- Biblioteca de materiais enviados pelo professor.
- Compartilhamento de aulas com outros professores.
- Sistema de turmas.

## Acessibilidade

- Tradução em tempo real para Libras.

- Modo escuro.
